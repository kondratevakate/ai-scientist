# Public Domain Compilation Scope

This project is prepared as a CC0 public-domain compilation of original review
work about AI Scientist systems and autonomous scientific discovery agents.

## Covered By CC0

The CC0 dedication covers original compilation work created for this project:

- source inventories and normalized registers
- PRISMA screening, deduplication, and protocol notes
- review matrices, taxonomies, and critique frameworks
- extraction forms and audit checklists
- repository metadata summaries written for this project
- search logs that describe queries, failures, and normalization decisions

## Not Covered By CC0

The CC0 dedication does not cover third-party material, including:

- cloned repositories under `open-source/`
- raw database/API exports under `searches/`
- copied abstracts, paper text, figures, screenshots, logos, or source code
- third-party names, trademarks, datasets, model weights, or benchmark files
- any content governed by upstream repository, publisher, database, or website
  terms

## Publication Rule

For a clean public release, publish the markdown compilation and metadata files
only. Do not publish local third-party snapshots or raw search exports unless
each item has been separately reviewed for redistribution rights.

Recommended GitHub state:

- include: `*.md`, `LICENSE`, `.gitignore`, and project metadata files
- exclude: `open-source/`
- exclude by default: `searches/`
- keep third-party repositories represented by URLs, commit hashes, and audit
  notes in `code-repos.md`

## Contributor Rule

Contributions should be original, source-traceable, and compatible with CC0.
Do not paste substantial third-party text into the compilation. Summarize,
cite, and link instead.

## Non-Legal Note

This file is a project governance note, not legal advice. If the compilation is
used for formal publication, institutional archiving, or commercial
redistribution, have the release reviewed by someone responsible for licensing.
